# 搭子计划 - 管理后台项目专属规则

## 项目定位

"搭子计划"管理后台。本项目基于 React + Ant Design 构建，**PC端WEB为主，兼容手机端**。

---

## OldSun 自动同步注意事项

OldSun 自动同步会将外部代码同步到本仓库，可能覆盖本地未提交的修改。

- **修改前**：务必先 `git commit` 提交
- **同步前**：确认没有未提交的修改
- **同步后**：检查 `git diff` 确认没有文件被覆盖
